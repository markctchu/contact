The design theme is configured for a **dark** interface, aiming for a **vibrant** aesthetic.

### Typography
The primary typeface for all text elements – headlines, body copy, and labels – is **Plus Jakarta Sans**, ensuring a consistent and modern typographic voice throughout the interface.

### Color Palette
The brand's core color palette is defined as follows:
- **Primary Color:** `#8B5CF6` (A distinctive purple)
- **Secondary Color:** `#2DD4BF` (A vibrant teal)
- **Tertiary Color:** `#FDE047` (A bright yellow accent)
- **Neutral Color:** `#121212` (A deep dark gray for backgrounds and surfaces)

### Shape and Form
UI elements feature **moderate** roundedness (level 2), striking a balance between modern softness and clean lines.

### Spacing
The layout employs **normal** spacing (level 2), providing a balanced and comfortable density for content and interactive elements.
